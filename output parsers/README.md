# Output Parsers

## What is this?
Output parsers convert raw model replies into useful Python data formats. This folder covers plain strings, JSON, Pydantic models, and simple structured schemas. In short: parsers make model output easier to use in code.

## Why does it exist?
- Use when your app needs predictable model output instead of free-form text.
- Use this README when you open this folder and want a quick overview before reading code.
