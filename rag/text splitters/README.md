# Text Splitters

## What is this?
Text splitters break large text into smaller chunks for embeddings and retrieval. This folder compares length-based and recursive splitting. In short: splitting makes documents easier to search and send to models.

## Why does it exist?
- Use before embeddings, vector stores, or RAG retrieval.
- Use this README when you open this folder and want a quick overview before reading code.
