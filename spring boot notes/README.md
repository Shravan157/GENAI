# Spring Boot Notes

## What is this?
This folder stores PDF notes about Spring Boot and related backend topics. These are reference documents, not GenAI scripts. In short: it is a PDF resource folder.

## Why does it exist?
- Use when you want quick access to the Spring Boot study material.
- Use this README when you open this folder and want a quick overview before reading code.
