# Console AI App

## What is this?
This folder contains a simple terminal chatbot that keeps conversation history. It shows how HumanMessage and AIMessage objects create memory across turns. In short: it is a minimal chat loop.

## Why does it exist?
- Use when you want to practice conversational memory in a command-line app.
- Use this README when you open this folder and want a quick overview before reading code.

## Files in this Folder

### Files
