# Cosine Similarity App

## What is this?
This folder compares a user query against documents using embeddings. It converts text into vectors, then ranks documents by cosine similarity. In short: it finds the closest matching document by meaning.

## Why does it exist?
- Use when you want semantic search basics without a vector database.
- Use this README when you open this folder and want a quick overview before reading code.
