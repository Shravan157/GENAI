# Document Loaders

## What is this?
Document loaders convert external sources into LangChain Document objects. This folder covers text, PDF, CSV, directory, and web loaders. In short: loaders bring outside knowledge into your GenAI app.

## Why does it exist?
- Use when your model needs content from files or websites.
- Use this README when you open this folder and want a quick overview before reading code.
